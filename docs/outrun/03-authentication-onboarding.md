OUTRUN V1 – AUTHENTICATION & THE ONBOARDING EXPERIENCE
________________________________________
OBJECTIVE
The onboarding experience is the single most important part of Outrun.
It must not feel like creating an account.
It must feel like hiring an AI Growth Consultant.
The user should never think:
"I'm filling in forms."
Instead they should think:
"This software is trying to understand my business."
The onboarding should create the first emotional attachment to Outrun.
Every question must have a purpose.
Do not ask unnecessary questions.

SUCCESS CRITERIA
The onboarding is successful if:
The user feels understood.
The user receives actionable insights.
The user experiences genuine value before seeing pricing.
The user believes Outrun has already saved them hours of work.
The user leaves onboarding confident about the next step in growing their business.
If any of these outcomes are not achieved, redesign the onboarding until they are.
