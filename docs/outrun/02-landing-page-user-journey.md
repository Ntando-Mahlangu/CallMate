LANDING PAGE RULES
Never use fake urgency.
Never use fake reviews.
Never use countdown timers.
Never use misleading claims.
Never overwhelm visitors with text.
Every section must answer:
Why should I trust Outrun?
How will Outrun help me grow?
What happens if I sign up today?
If a section does not answer one of these questions, redesign or remove it.
________________________________________
DEFINITION OF SUCCESS
The landing page succeeds if visitors understand the product within 30 seconds, trust the platform within 2 minutes, and feel excited to begin onboarding before they reach the bottom of the page.
