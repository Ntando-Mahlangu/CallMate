OBJECTIVE
The dashboard is not a reporting tool.
The dashboard is Mission Control.
The user should never wonder:
"What should I do now?"
Within five seconds of opening Outrun they should know exactly what action will have the biggest impact on growing their business.
The dashboard must reduce decision fatigue.
Everything on this page must answer one question:
What is the next best action this business should take?
