OUTRUN V1 – AI GROWTH BLUEPRINT ENGINE
________________________________________
OBJECTIVE
The AI Growth Blueprint is the signature feature of Outrun.
It must be so valuable that users would willingly pay for it on its own.
The Blueprint is not a report.
It is a strategic action plan tailored to the user's business.
The user should think:
"This understands my business better than I expected."
Every recommendation must be personalized, actionable, transparent, and clearly distinguish facts from AI-generated recommendations.
________________________________________
INPUT SOURCES
The AI should combine information from:
Onboarding answers
Website analysis (if available)
Business description
Industry
Target market
Campaign performance
User goals
AI Memory
Future connected integrations (CRM, email, analytics, etc.)
Never invent facts.
If information is missing, explicitly state that the recommendation is based on limited information.
